# Examples

This directory contains practical examples demonstrating how to use objectory in various scenarios.

## Available Examples

1. **basic_factory.py** - Basic usage of the universal factory function
2. **abstract_factory_example.py** - Using the AbstractFactory metaclass
3. **registry_example.py** - Using the Registry class
4. **plugin_system.py** - Building a plugin system with objectory
5. **configuration_loader.py** - Loading objects from configuration files

## Running Examples

Each example can be run directly with Python:

```bash
python examples/basic_factory.py
```

Or from the project root:

```bash
python -m examples.basic_factory
```

## Requirements

All examples require objectory to be installed:

```bash
pip install objectory
```

Some examples may have additional dependencies, which are noted in the example files.
